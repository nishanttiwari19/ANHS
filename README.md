# Adarsh Narmada Vidhyalaya School Website

A premium school website built with Java (Spring Boot), SQL (H2), and modern HTML/CSS/JS.

## Features
- **Dynamic Notifications**: Admin can post latest updates via the dashboard.
- **Premium UI**: Modern, responsive design with a Navy/Yellow color palette.
- **Admin Portal**: Secure login and management dashboard.
- **Real-time Updates**: Homepage updates instantly when a new notification is posted.

## Tech Stack
- **Backend**: Java 17+, Spring Boot 3.2
- **Database**: H2 (SQL)
- **Frontend**: HTML5, Vanilla CSS3, JavaScript

## How to Run
1. Ensure you have **Java 17+** installed.
2. If you have Maven: `mvn spring-boot:run`
3. If you don't have Maven, you can use the Maven Wrapper (if provided) or install it.
4. Access the site at: `http://localhost:8080`
5. Admin Login: `admin` / `admin123`
